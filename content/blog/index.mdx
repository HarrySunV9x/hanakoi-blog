# 前言

记录值得收藏的开源库，方便取用。

# AI

[ChatTTS](https://github.com/2noise/ChatTTS)：专门为对话场景设计的文本转语音模型，优势在于短文本语音效果很好。

[ComfyUI](https://github.com/comfyanonymous/ComfyUI)：基于流的文生图工具

[Deep-Live-Cam](https://github.com/hacksider/Deep-Live-Cam)：摄像头AI替换人脸

[gradio](https://github.com/gradio-app/gradio): 快速构建llm前端应用

[supervision](https://github.com/roboflow/supervision): 计算机视觉检测工具

# 数据库

[pocketbase](https://github.com/pocketbase/pocketbase): 基于golang的简易数据库后端

# Android 开发

[compose-samples](https://github.com/android/compose-samples): compose示例

# C++库

[spdlog](https://github.com/gabime/spdlog): 日志打印库，好用

[googletest](https://github.com/google/googletest)：C++自动化测试工具

# 其他

[core](https://github.com/home-assistant/core): Home Assistant库

[Reactive-Resume](https://github.com/AmruthPillai/Reactive-Resume): 简历设计

[drawio-desktop](https://github.com/jgraph/drawio-desktop): 画图工具

[immich](https://github.com/immich-app/immich): 图片视频管理
